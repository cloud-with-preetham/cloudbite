# Auth Service

The Auth Service handles user registration, login, JWT-based authentication, and role-based access control for the CloudBite platform.

## Responsibilities

- Register new users
- Authenticate existing users
- Generate JWT tokens
- Manage user roles
- Provide authentication health checks

## Planned Roles

- CUSTOMER
- RESTAURANT_OWNER
- DELIVERY_PARTNER
- ADMIN

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT
- Docker
